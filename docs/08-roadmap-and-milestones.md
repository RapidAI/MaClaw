# MaClaw Roadmap And Milestones

## 1. Priority Order

1. Claude hosting on Desktop
2. Hub compressed-state pipeline
3. PWA read-only pocket view
4. PWA control loop
5. Hub auth/admin completion
6. Hub Center
7. Pocket app

## 2. Milestone 1

Claude remote loop:

- Desktop hosts Claude
- Desktop generates compressed state
- Hub receives and stores that state
- debug APIs inspect it

## 3. Milestone 2

Compressed-state quality:

- stable summary
- useful important events
- readable preview

## 4. Milestone 3

Hub self-auth:

- admin setup
- desktop enrollment
- email-confirm login for PWA

## 5. Milestone 4

PWA read-only:

- machine list
- session list
- session detail

## 6. Milestone 5

PWA control:

- send input
- interrupt
- kill

## 7. Milestone 6

Hub Center:

## 8. Milestone 7

Pocket app:

- PWA launch

## 9. Milestone 8

Governance and collaboration:
