# MaClaw iOS Shell

This iOS shell project was generated by MaClaw's Go-based mobile shell generator.

## Defaults

- App name: MaClaw
- Bundle ID: tech.rapidai.hubshell
- Start URL: bootstrap
- Hub Center: http://hubs.mypapers.top:9388

## Output

All iOS simulator artifacts are written into `../dist/`.

Current filenames:

- Debug simulator app: `maclaw-ios-simulator.app`
- Release simulator app: `maclaw-ios-simulator-release.app`
- DerivedData root: `ios-deriveddata/`

## Scripts

- `build_ios_simulator.sh`
  Builds the debug simulator app and copies the `.app` bundle into `../dist/`.
- `build_ios_simulator_release.sh`
  Builds the release simulator app and copies the `.app` bundle into `../dist/`.

## Common Flows

Debug simulator app:

```bash
sh ./build_ios_simulator.sh
```

Release simulator app:

```bash
sh ./build_ios_simulator_release.sh
```

## Xcode

You can also open `RapidAIHubShell.xcodeproj` in Xcode. The shell scripts simply wrap `xcodebuild` and keep simulator outputs collected under `mobile/dist/`.
